# The SE AI Toolkit

A hands-on guide to three AI tools for Salesforce SEs: **Claude Code**, **Cursor**, and **Saleo**.

I built this because our team has access to these tools but no single place that explains when to use each one, how they fit together, and what good looks like in practice. The guide is structured like a Trailhead module — it starts with the basics, builds up to real workflows, and covers the full loop from building a demo integration to presenting it to a prospect.

It's not a repo you need to clone. The guide is hosted at **[https://jarteaga-sf.github.io/se-ai-toolkit/](https://jarteaga-sf.github.io/se-ai-toolkit/)** — just open the link.
